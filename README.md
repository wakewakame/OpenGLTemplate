# OpenGLTemplate
OpenGLとOpenALを用いた開発環境の雛形です。

# ビルド方法&実行方法
gitとcmakeが必要になります。  

```
$ git clone https://github.com/wakewakame/OpenGLTemplate
$ cd OpenGLTemplate
$ git submodule update --init --recursive
$ mkdir build && cd build
$ cmake ..
$ make
$ ./OpenGLTemplate
```
