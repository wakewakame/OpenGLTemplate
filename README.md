# jummmp
大学3年生の頃に作った作品です。  
自分の実力を試すため、自分が使える技術をできるだけ盛り込みました。  

# ビルド方法&実行方法
gitとcmakeが必要になります。  
```
$ git clone https://github.com/wakewakame/jummmp
$ cd jummmp
$ git submodule update --init --recursive
$ mkdir build && cd build
$ cmake ..
$ make
$ ./jummmp
```
